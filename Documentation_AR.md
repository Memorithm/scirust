# توثيق SciRust 🦀

مرحباً بكم في توثيق **SciRust**، إطار عمل للتعلم العميق والحوسبة العلمية مكتوب بالكامل بلغة **Rust الصافية**.

## 1. ما هو SciRust؟

SciRust هو منصة للبحث والتطوير في مجال الذكاء الاصطناعي. على عكس العديد من الأدوات الأخرى (مثل PyTorch أو TensorFlow) التي تعتمد على مكتبات C++ أو Python معقدة، تم بناء SciRust من الألف إلى الياء باستخدام Rust.

**لماذا هذا مهم؟**
- **شفافية كاملة**: يمكنك قراءة كل سطر من كود الحوسبة، من طبقة الشبكة إلى النواة الرياضية.
- **الأمان والموثوقية**: يستفيد من ضمانات الذاكرة والأمان التي توفرها لغة Rust.
- **الاستقلالية**: لا يتطلب أي تبعيات خارجية معقدة (FFI).

## 2. الفلسفة والمزايا الرئيسية

لا يحاول SciRust استبدال عمالقة الصناعة، ولكنه يقدم نهجاً مختلفاً يركز على **الثقة** و **قابلية إعادة الإنتاج**.

### الحتمية بت-بمقابل-بت (Bit-for-Bit Determinism)
في العديد من أطر العمل، يمكن أن يؤدي تشغيل نفس الحساب مرتين إلى نتائج مختلفة قليلاً (بسبب التوازي). يضمن SciRust **الحتمية بت-بمقابل-بت**: ستكون النتيجة متطابقة تماماً، بغض النظر عن عدد المعالجات المستخدمة. هذا أمر بالغ الأهمية لقابلية التدقيق.

### قابلية التدقيق (Auditability)
بما أن كل شيء مكتوب بلغة Rust، فمن السهل التحقق من أن الكود يفعل بالضبط ما يقوله. لا يوجد "صندوق أسود" برمجي.

### أوراكل التحقق (Validation Oracles)
يتم التحقق من كل دالة رياضية في SciRust مقابل "أوراكل تحقق" (مرجع موثوق). نحن لا نفترض أن النتيجة صحيحة؛ بل نقيسها.

## 3. مجالات التطبيق

يعد SciRust مفيداً بشكل خاص في المجالات التي تكون فيها الدقة والأمان والبصمة البرمجية الصغيرة أمراً حيوياً:

- **الأنظمة المدمجة (Edge AI)**: بفضل بصمته المنخفضة وقدرات التكميم (تقليل حجم النموذج)، فإنه يعمل بشكل مثالي على الأجهزة الصغيرة.
- **القطاعات المنظمة (الطيران، الطب، التمويل)**: حيث يجب أن يكون كل قرار يتخذه الذكاء الاصطناعي قابلاً لإعادة الإنتاج والتفسير لأسباب تتعلق بالسلامة أو الامتثال.
- **البحث العلمي**: لاكتشاف القوانين الرياضية من البيانات من خلال الانحدار الرمزي.
- **تدقيق الأمان**: للشركات التي تحتاج إلى اعتماد سلسلة الحوسبة الخاصة بها بالكامل.

## 4. ما يمكنك تحقيقه

يغطي SciRust مجموعة واسعة من التقنيات الحديثة:

- **التعلم العميق**: بناء الشبكات العصبية (MLP, CNN, Transformers) مع التفاضل التلقائي (autograd).
- **الانحدار الرمزي**: اكتشاف الصيغ الرياضية (مثلاً `f(x) = sin(x) + x^2`) من الملاحظات.
- **التحسين التطوري**: استخدام الخوارزميات المستوحاة من الطبيعة (مثل NSGA-II) لحل المشكلات المعقدة.
- **التكميم int8**: تقليص حجم النموذج بمقدار 4 مرات ليناسب المعالجات الصغيرة دون فقدان الدقة.
- **تسريع GPU**: تسخير قوة بطاقات الرسومات عبر WebGPU (wgpu) أو NVIDIA Tensor Cores (cuBLAS).

## 5. دليل الأوامر

يتم استخدام SciRust بشكل أساسي عبر الطرفية باستخدام `cargo` ، وهي أداة Rust القياسية.

### التثبيت
أضف هذا إلى ملف `Cargo.toml` الخاص بك:
```toml
[dependencies]
scirust-core = { path = "..." }
```

### الترجمة والاختبار
- **فحص المشروع**: `cargo check --workspace`
- **تشغيل جميع الاختبارات** (أكثر من 250 اختباراً للتحقق من إطار العمل): `cargo test --workspace`
- **الترجمة في وضع التحسين** (موصى به للذكاء الاصطناعي): `cargo build --release`
- **تمكين دعم GPU**: أضف `--features wgpu` إلى أوامرك.

### أمثلة التشغيل
- **تدريب MNIST (الأرقام المكتوبة بخط اليد)**:
  ```bash
  cargo run --example mnist_classifier --release
  ```
- **عرض ضغط Transformer**:
  ```bash
  cargo run -p transformer_compress --release
  ```
- **اختبار أداء ضرب المصفوفات**:
  ```bash
  cargo run -p scirust-core --example bench_matmul --release
  ```

## 7. الخاتمة

SciRust هو إطار العمل المفضل لأولئك الذين يعطون الأولوية لـ **الفهم** و **الصرامة** على السرعة الخام أو سهولة Python. إنه أداة قوية لبناء ذكاء اصطناعي جدير بالثقة، من البحث إلى الأنظمة المدمجة.
